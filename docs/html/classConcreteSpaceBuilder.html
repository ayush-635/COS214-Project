Detailed Description

Concrete builder for constructing plantable spaces.

Implements the SpaceBuilder interface to create PlantableArea structures with PlanterBoxes and nested PlanterBoxCollections. Provides methods to:

  • Add individual planter boxes
  • Add nested sub-collections
  • Add default components (2 boxes)

Used by NurseryManager (Director) to construct small, medium, and large spaces.

Note
Part of the Builder design pattern
