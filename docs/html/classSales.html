◆ findMatchingPlant()

std::string Sales::findMatchingPlant ( bool  outside,
bool  lowLight,
bool  lowWater,
bool  brightColour,
bool  lowCare 
)

Finds a plant matching the given preference flags. Workflow:

  1. Query Inventory for all plant names
  2. For each plant that has stock: a. Get prototype and PlantData b. Check if PlantData matches preference flags c. Check stock > 0
  3. If match found, return plant name and stock
  4. If no matches found, return fallback message.
    Parameters
    outsideWhether the plant should be suitable for outside.
    lowLightWhether the plant should be suitable for low light.
    lowWaterWhether the plant should be suitable for low water.
    brightColourWhether the plant should have bright colors.
    lowCareWhether the plant should require low maintenance.
    Returns
    The name of the matching plant or a fallback message.
