Detailed Description

Base class for all staff members in the simulation. concrete classes are: 

@class StaffMember

- Gardner

  • Sales
  • Cashier
  • Pathologist This class tracks how much time a staff member has spent on duties. The standard duty pipeline is:
  1. addTime(time)
  2. tick(time) - role-specific time update
  3. browse() - optional browsing action
  4. receivePreference(pref) - optional preference reception
  5. sendAdvice(advice) - optional advice sending Concrete subclassses must implement tick().
