Detailed Description

Composite class representing a collection of planter boxes.

This class can contain multiple PlanterBox objects and allows for operations to be performed on the collection as a whole or delegated to individual boxes.

Stores a depth variable to see nesting of the composites in the collections Stores a maximum depth and maximum number of boxes allowed
