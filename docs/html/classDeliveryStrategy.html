Detailed Description

Abstract base class for delivery strategies.

This class defines the interface for all concrete delivery strategy implementations. Each concrete strategy (StandardDelivery, ExpressDelivery, PickupDelivery) implements the deliver() method differently to provide various delivery options with different costs and delivery types.

Note
This is part of the Strategy design pattern
