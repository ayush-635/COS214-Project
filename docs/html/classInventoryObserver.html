Detailed Description

Observer that monitors stock changes in the Inventory. When stock levels change, it updates its internal cache and notifies customers if new stock arrives. This observer keeps a small cache of stock levels for plants it is interested in.

  • key: Plant* (pointer to the plant)
  • value: int (current stock level) adds notification functionality when stock goes from 0 to >0.
