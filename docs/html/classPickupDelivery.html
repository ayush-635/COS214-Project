Detailed Description

Concrete strategy for customer pickup service.

Implements the cost-free customer pickup option with:

  • Delivery time: 2-4 hours (order preparation)
  • Cost: R0.00 (FREE - no delivery charge)
  • Pickup location: Main Nursery - 123 Garden St
  • Pickup hours: Mon-Sat 8AM-6PM, Sunday 9AM-4PM
  • Requires ID and order confirmation

Features fancy formatted output with box-drawing characters and emojis to create a customer pickup receipt display.

Note
Part of the Strategy design pattern
This is the most economical option (FREE) but requires customer transportation
